# Voice & Tone Guide
## Bild-Style German Content

**Ziel:** Unterhaltsam, direkt, emotional - wie ein Gespräch mit einem Freund, der Ahnung hat.

---

## Die 10 Regeln

### 1. Kurze Sätze. Punkt.
Maximal 12-15 Wörter pro Satz. Lieber zwei kurze als ein langer.

```
❌ "Es ist wichtig zu beachten, dass die regelmäßige Verwendung von zu heißem Wasser beim Duschen dazu führen kann, dass die natürlichen Öle der Kopfhaut entfernt werden."

✅ "Heißes Wasser ist ein Problem. Es zieht die natürlichen Öle aus deiner Haut. Zurück bleibt: Trockenheit."
```

### 2. Du, nicht man
Direkte Ansprache. Immer.

```
❌ "Man sollte darauf achten..."
✅ "Achte darauf..."

❌ "Es empfiehlt sich..."
✅ "Mach das:"

❌ "Man kennt das Problem..."
✅ "Du kennst das."
```

### 3. Fragen stellen
Fragen aktivieren den Leser. Sie fühlen sich angesprochen.

```
"Klingt bekannt?"
"Und jetzt?"
"Das Ergebnis?"
"Warum ist das so?"
"Hast du das schon mal probiert?"
```

### 4. Emotionen benennen
Sag, was der Leser fühlt - bevor er es selbst sagt.

```
"Frustrierend."
"Nervig, oder?"
"Das macht wahnsinnig."
"Endlich eine Antwort."
"Erleichterung."
```

### 5. Cliffhanger & Teaser
Neugier wecken, bevor du die Antwort gibst.

```
"Aber da gibt es eine Sache, die fast niemand weiß..."
"Hier kommt der Teil, den dir keiner erzählt."
"Spoiler: Es liegt nicht am Shampoo."
"Die Ursache? Überraschend."
```

### 6. Zahlen statt vage Wörter
Konkret schlägt abstrakt.

```
❌ "mehrere Ursachen"
✅ "7 Ursachen"

❌ "eine Weile einwirken lassen"
✅ "3 Minuten einwirken lassen"

❌ "regelmäßig"
✅ "alle 2-3 Tage"
```

### 7. Aufzählungen mit Punch
Nicht einfach auflisten - jeder Punkt hat Energie.

```
❌
- Shampoo wechseln
- Weniger waschen
- Kälteres Wasser verwenden

✅
- Shampoo? Gewechselt.
- Seltener gewaschen? Check.
- Kälteres Wasser? Probiert.
- Und trotzdem: Das Jucken bleibt.
```

### 8. Betonung durch Struktur
Wichtige Punkte isolieren. Wirken lassen.

```
"Nicht das Shampoo. Nicht die Temperatur. Das Wasser selbst."

"Das. Ist. Das. Problem."

"Und dann? Dann juckt es wieder."
```

### 9. Umgangssprache (dosiert)
Nicht übertreiben, aber menschlich klingen.

**Erlaubt:**
- "Echt jetzt?"
- "Klingt verrückt, ist aber so."
- "Das Ding ist..."
- "Kurz gesagt:"
- "Hand aufs Herz:"
- "Keine Sorge."
- "Ganz ehrlich?"

**Vermeiden:**
- Jugendsprache (cringe, sus, etc.)
- Zu viel Slang
- Dialekt

### 10. Ende mit Empowerment
Der Leser soll sich schlauer fühlen als vorher.

```
"Jetzt weißt du, woran es liegt."
"Probier's aus - du hast nichts zu verlieren."
"Und wenn nichts hilft? Dann weißt du jetzt, wo du ansetzen kannst."
```

## Magic Words

| Wort | Wirkung | Beispiel |
|------|---------|----------|
| **Du** | Direkt, persönlich | "Du kennst das." |
| **Jetzt** | Dringlichkeit | "Jetzt wird's interessant." |
| **Wirklich** | Authentizität | "Was wirklich hilft." |
| **Endlich** | Erlösung | "Endlich eine Erklärung." |
| **Geheim/Versteckt** | Neugier | "Die versteckte Ursache." |
| **Einfach** | Zugänglichkeit | "Die einfache Lösung." |
| **Sofort** | Schnelle Hilfe | "Soforthilfe." |
| **Überraschend** | Aufmerksamkeit | "Die überraschende Wahrheit." |
| **Niemand** | Exklusivität | "Was niemand erzählt." |
| **Kostenlos** | Niedrige Hürde | "Und das Beste? Kostet nichts." |

## Satz-Templates

### Einstieg / Hook
```
"Du kennst das: [Problem beschreiben]."
"[Frage]? Dann lies weiter."
"Das hier wird dir bekannt vorkommen."
"Mal ehrlich: [Problem]."
```

### Überleitung
```
"Aber jetzt wird's interessant."
"Hier kommt der Teil, den dir keiner erzählt."
"Und jetzt? Jetzt wird's konkret."
"Schauen wir uns das genauer an."
```

### Reveal / Aha-Moment
```
"Die Wahrheit? [Überraschung]."
"Spoiler: Es liegt nicht an [offensichtliche Vermutung]."
"Das Problem ist nicht [X]. Es ist [Y]."
"Klingt absurd? Ist aber so."
```

### Lösung einleiten
```
"Die gute Nachricht:"
"Was du tun kannst:"
"Und das Beste? Es ist einfach."
"Hier ist, was wirklich hilft."
```

### Abschluss
```
"Jetzt weißt du Bescheid."
"Probier's aus."
"Und wenn das nicht hilft - du weißt jetzt, woran es liegen könnte."
"Viel Erfolg."
```

## Formatierung

### Absätze
- Maximal 3-4 Sätze pro Absatz
- Viel Weißraum
- Mobile-first denken

### Listen
- Nummeriert wenn Reihenfolge wichtig
- Bullet Points für gleichwertige Items
- Kurze Punkte (1 Zeile ideal)

### Hervorhebungen
- **Fett** für wichtige Begriffe
- Sparsam einsetzen
- Keine GROSSBUCHSTABEN für Betonung

### Zwischenüberschriften
- Fragen funktionieren gut
- Oder: Neugier wecken
- Keywords einbauen, aber natürlich

## Emotionale Struktur

Jeder Artikel folgt diesem Bogen:

```
1. ERKENNUNG
   "Du kennst das..." → Leser fühlt sich verstanden

2. FRUSTRATION
   "Nichts hilft..." → Validierung des Problems

3. NEUGIER
   "Aber da gibt es etwas..." → Spannung aufbauen

4. ÜBERRASCHUNG
   "Das Wasser selbst!" → Aha-Moment

5. HOFFNUNG
   "Die Lösung ist einfacher als gedacht..." → Erleichterung

6. EMPOWERMENT
   "Jetzt weißt du, was zu tun ist." → Handlungsfähigkeit
```

## Verbotene Phrasen

| ❌ Nicht schreiben | ✅ Stattdessen |
|-------------------|----------------|
| "Es ist allgemein bekannt, dass..." | "Fakt ist:" |
| "Studien haben gezeigt, dass..." | "Was die Forschung sagt:" |
| "In der heutigen Zeit..." | Streichen. |
| "Wie bereits erwähnt..." | Streichen oder neu formulieren. |
| "Abschließend lässt sich sagen..." | "Kurz gesagt:" |
| "Es sollte beachtet werden..." | "Wichtig:" |
| "Im Folgenden wird erläutert..." | Einfach erläutern. |
| "Grundsätzlich..." | Streichen oder konkret werden. |
| "Diesbezüglich..." | Umformulieren. |
| "Selbstverständlich..." | Streichen. |

## Beispiel: Vorher/Nachher

### Thema: Heißes Wasser trocknet Kopfhaut aus

**❌ Vorher (generisch, langweilig):**
> Eine häufige Ursache für juckende Kopfhaut nach dem Duschen ist die Verwendung von zu heißem Wasser. Heißes Wasser kann dazu führen, dass die natürlichen Öle der Kopfhaut entfernt werden, was wiederum zu Trockenheit und Reizungen führen kann. Es empfiehlt sich daher, die Wassertemperatur zu reduzieren und lauwarmes Wasser zu verwenden.

**✅ Nachher (Bild-Style):**
> Zu heiß geduscht? Deine Kopfhaut merkt sich das.
>
> Heißes Wasser fühlt sich gut an - aber es zieht die natürlichen Öle aus deiner Haut. Zurück bleibt: Trockenheit. Juckreiz. Und das nach jeder Dusche.
>
> Die Lösung? Lauwarm duschen. Klingt unsexy, wirkt aber.
>
> Schnelltest: Wenn dein Spiegel komplett beschlägt, ist das Wasser zu heiß.

---

### Thema: Kalk im Wasser

**❌ Vorher:**
> Kalk im Leitungswasser kann Ablagerungen auf der Kopfhaut hinterlassen, welche zu Irritationen führen können. Dies ist insbesondere in Regionen mit hoher Wasserhärte ein häufiges Problem.

**✅ Nachher:**
> Hier kommt der Teil, den dir keiner erzählt: Das Problem ist nicht dein Shampoo. Es ist das Wasser.
>
> Klingt absurd? Ist aber so.
>
> In vielen deutschen Städten ist das Leitungswasser so kalkhaltig, dass es einen unsichtbaren Film hinterlässt. Auf deinen Armaturen. Auf deinem Wasserkocher. Und ja - auf deiner Kopfhaut.
>
> Du wäschst dir die Haare mit dem sanftesten Shampoo der Welt. Dann spülst du es aus - mit Wasser, das selbst Rückstände hinterlässt.
>
> Das Shampoo ist weg. Die Mineralien bleiben.
