# Lebensgunst Blog Blueprint

> Voice: "Bild-Style" — unterhaltsam, direkt, emotional. Wie ein Gespräch mit einem Freund, der Ahnung hat.

---

## The Formula

### Article Structure

```
1. HOOK (emotional, pain-first)
2. TL;DR BOX (direkt nach Hook)
3. INHALTSVERZEICHNIS
4. FRUSTRATION BUILD (alles probiert, nichts hilft)
5. THE REVEAL (das Wasser!)
6. SCIENCE SECTION (was Kalk/Chlor anrichten)
7. SELF-TEST (Leser checkt selbst)
8. DIE LÖSUNG (Duschfilter = Hauptlösung)
9. WANN ZUM ARZT (medical disclaimer)
10. FAQ SECTION (für SEO/Featured Snippets)
11. EMPOWERMENT CLOSE
```

---

## Voice Rules (Bild-Style)

### 1. Kurze Sätze. Punkt.
Max 12-15 Wörter. Lieber zwei kurze als ein langer.

```
❌ "Es ist wichtig zu beachten, dass die regelmäßige Verwendung von zu heißem Wasser beim Duschen dazu führen kann, dass die natürlichen Öle der Kopfhaut entfernt werden."

✅ "Heißes Wasser ist ein Problem. Es zieht die natürlichen Öle aus deiner Haut. Zurück bleibt: Trockenheit."
```

### 2. Du, nicht man
```
❌ "Man sollte darauf achten..."
✅ "Achte darauf..."

❌ "Es empfiehlt sich..."
✅ "Mach das:"
```

### 3. Fragen stellen
```
"Klingt bekannt?"
"Und jetzt?"
"Das Ergebnis?"
"Warum ist das so?"
```

### 4. Emotionen benennen
```
"Frustrierend."
"Nervig, oder?"
"Das macht wahnsinnig."
"Endlich eine Antwort."
```

### 5. Cliffhanger & Teaser
```
"Aber da gibt es eine Sache, die fast niemand weiß..."
"Hier kommt der Teil, den dir keiner erzählt."
"Spoiler: Es liegt nicht am Shampoo."
```

### 6. Zahlen statt vage Wörter
```
❌ "mehrere Ursachen"
✅ "7 Ursachen"

❌ "regelmäßig"
✅ "alle 2-3 Tage"
```

### 7. Aufzählungen mit Punch
```
❌
- Shampoo wechseln
- Weniger waschen

✅
- Shampoo? Gewechselt.
- Seltener gewaschen? Check.
- Kälteres Wasser? Probiert.
- Und trotzdem: Das Jucken bleibt.
```

### 8. Betonung durch Struktur
```
"Nicht das Shampoo. Nicht die Temperatur. Das Wasser selbst."
"Das. Ist. Das. Problem."
```

### 9. Umgangssprache (dosiert)
**Erlaubt:**
- "Echt jetzt?"
- "Klingt verrückt, ist aber so."
- "Das Ding ist..."
- "Hand aufs Herz:"

**Vermeiden:**
- Jugendsprache (cringe, sus)
- Zu viel Slang
- Dialekt

### 10. Ende mit Empowerment
```
"Jetzt weißt du, woran es liegt."
"Probier's aus – du hast nichts zu verlieren."
```

## Magic Words

| Wort | Wirkung |
|------|---------|
| **Du** | Direkt, persönlich |
| **Jetzt** | Dringlichkeit |
| **Wirklich** | Authentizität |
| **Endlich** | Erlösung |
| **Geheim/Versteckt** | Neugier |
| **Einfach** | Zugänglichkeit |
| **Sofort** | Schnelle Hilfe |
| **Überraschend** | Aufmerksamkeit |
| **Niemand** | Exklusivität |

## Emotional Arc

```
1. ERKENNUNG      → "Du kennst das..." (Leser fühlt sich verstanden)
2. FRUSTRATION    → "Nichts hilft..." (Validierung)
3. NEUGIER        → "Aber da gibt es etwas..." (Spannung)
4. ÜBERRASCHUNG   → "Das Wasser selbst!" (Aha-Moment)
5. HOFFNUNG       → "Die Lösung ist einfacher..." (Erleichterung)
6. EMPOWERMENT    → "Jetzt weißt du, was zu tun ist." (Handlungsfähigkeit)
```

## Verbotene Phrasen

| ❌ Nicht schreiben | ✅ Stattdessen |
|-------------------|----------------|
| "Es ist allgemein bekannt, dass..." | "Fakt ist:" |
| "Studien haben gezeigt, dass..." | "Was die Forschung sagt:" |
| "In der heutigen Zeit..." | Streichen. |
| "Abschließend lässt sich sagen..." | "Kurz gesagt:" |
| "Es sollte beachtet werden..." | "Wichtig:" |
| "Grundsätzlich..." | Streichen oder konkret werden. |
| "Selbstverständlich..." | Streichen. |

## Required Sections

### TL;DR Format (direkt nach Hook)
```markdown
> **Kurz & knapp:** [Problem] liegt meist an [Ursache].
> Die Lösung: [Duschfilter pitch]. Lies weiter für Details.
```

### Inhaltsverzeichnis
Nach TL;DR. 8-12 Sections mit Anchor Links.

### Duschfilter = HAUPTLÖSUNG
Nicht "ein Tipp unter vielen", sondern DIE Lösung. Andere Tipps sind "ergänzend".

### "Wann zum Arzt" Section
IMMER enthalten:
- Keine Diagnosen stellen
- Keine Heilversprechen
- Bei Krankheiten: "kann helfen", nicht "heilt"

### FAQ Section (am Ende)
Für Featured Snippets + Schema. 3-5 Fragen im Q&A Format.

---

## Formatting Rules

### Nur `-` verwenden
**Nie `–` oder `—`**. Vor Publish: Suchen & Ersetzen.

### Absätze
- Max 3-4 Sätze
- Viel Weißraum
- Mobile-first

### Hervorhebungen
- **Fett** für wichtige Begriffe
- Sparsam
- Keine GROSSBUCHSTABEN

---

## No Duplicate Content

Jeder Post braucht:
- Andere Struktur als existierende Posts
- Mindestens 3 unique Sections
- Andere Beispiele/Analogien
- Andere Produkt-Empfehlungen

---

## SEO-Struktur

- **FAQ-Section** für Featured Snippets
- **Quotable definitions** - klare Sätze die Google zitieren kann
- **Question-Answer Style** - Fragen als H2/H3
- **Key Takeaways** - wichtige Punkte fett oder als Liste
- **Interne Links** - zu anderen lebensgunst Artikeln

---

## Autorität zeigen

- Spezifische Produktnamen/Modelle nennen
- Zahlen und Fakten (°dH, Temperaturen, Prozente)
- "Warum" erklären, nicht nur "Was"

## Satz-Templates

### Einstieg / Hook
```
"Du kennst das: [Problem beschreiben]."
"[Frage]? Dann lies weiter."
"Das hier wird dir bekannt vorkommen."
```

### Überleitung
```
"Aber jetzt wird's interessant."
"Hier kommt der Teil, den dir keiner erzählt."
"Schauen wir uns das genauer an."
```

### Reveal / Aha-Moment
```
"Die Wahrheit? [Überraschung]."
"Spoiler: Es liegt nicht an [offensichtliche Vermutung]."
"Das Problem ist nicht [X]. Es ist [Y]."
```

### Lösung einleiten
```
"Die gute Nachricht:"
"Was du tun kannst:"
"Hier ist, was wirklich hilft."
```

### Abschluss
```
"Jetzt weißt du Bescheid."
"Probier's aus."
"Viel Erfolg."
```

## Hijack Topics

Target expensive products/trends where water quality is the hidden factor:

**Hair:**
- Dyson Airwrap / Supersonic ✅ (done)
- Olaplex not working
- Keratin treatment fading fast
- Hair color not holding
- Curly girl method failing
- Locken halten nicht

**Skin:**
- Expensive skincare not absorbing
- Adult acne despite good routine
- Dry skin despite moisturizing
- Eczema flare-ups

**Lifestyle:**
- Moving to Germany (expat shock)
- New apartment, suddenly bad hair/skin
- "My hair was better in [other country]"

## Pre-Publish Checklist

- [ ] Keine `–` im Text (nur `-`)
- [ ] TL;DR nach Intro vorhanden
- [ ] Inhaltsverzeichnis vorhanden
- [ ] FAQ-Section vorhanden
- [ ] Keine Überschneidung mit anderen Posts
- [ ] Duschfilter prominent positioniert (nicht nur "ein Tipp")
- [ ] "Wann zum Arzt" Section vorhanden
- [ ] Interne Links gesetzt
- [ ] Kurze Sätze (max 15 Wörter)
- [ ] "Du" statt "man"
- [ ] Zahlen statt vage Begriffe
